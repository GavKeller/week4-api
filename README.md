# Apparel Order API

## Description

This project is a basic Express API that tracks apparel orders. The API has different routes that return either regular text or JSON data. My custom data is a list of clothing orders with things like the customer name, item, size, and order status. This project helped me practice making routes and sending responses from a backend server.

## Available Routes

### /

This route shows a simple welcome message for the API.

### /about

This route explains what the API is used for.

### /status

This route shows that the API is running.

### /users

This route returns a small list of users in JSON format from the Week 4 demo.

### /orders

This route returns the apparel order data in JSON format.

### /info

This route shows basic information about the API, like the name, version, and author.

### /contact

This route returns basic contact information for the API.
